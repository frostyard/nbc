## nbc

A bootc container installer for physical disks

### Synopsis

nbc is a tool for installing bootc compatible containers to physical disks.
It automates the process of preparing disks and deploying bootable container images.

### Options

```
  -n, --dry-run   dry run mode (no actual changes)
  -h, --help      help for nbc
      --json      output in JSON format for machine-readable output
  -v, --verbose   verbose output
```
